This repository contains projects, assignments, and explorations for the Solana Fellowship.

- **`mini-solana-validator`**: An in-memory, single-node Solana-compatible JSON-RPC server designed to process real Solana transactions, supporting the System Program, SPL Token Program, and Associated Token Account Program.
- **`sol-launchpad`**: A modular backend application (built with Express.js and Prisma) serving as a launchpad for Solana-related functionalities.
